# LoFaCa: Logical Fallacy Classification

Identifying common logical fallacies in news comments using the CoCoLoFa dataset.

## Project Structure

- `data/`: Contains the CoCoLoFa dataset splits (Train, Dev, Test).
- `analyze_data.py`: Script for data distribution analysis.
- `exp.ipynb`: Notebook for experimentation.

## Dataset

CoCoLoFa contains 7,706 comments for 648 news articles, labeled for fallacy presence and type across 8 categories.
